#  Password Validation Tool

##  Project Description

This project is a Python-based Password Validation Tool that checks whether a given password meets specific security requirements. It also includes an additional feature to evaluate the strength of the password.

---

##  Objectives

* To validate passwords using string operations
* To enforce security rules like length, digits, and special characters
* To handle incorrect inputs using exception handling
* To provide password strength analysis (extra feature)

---

##  Features

*  Minimum length validation (8 characters)
*  Checks for uppercase and lowercase letters
*  Ensures at least one digit
*  Ensures at least one special character
*  Password strength indicator (Weak / Medium / Strong)
*  User-friendly interaction with multiple attempts

---

## Technologies Used

* Python
* Built-in string module

---

##  How to Run

1. Install Python (if not installed)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the file:

```bash
python project.py
```
